# Valera

![Unit Tests & Lint](https://github.com/GreyWayfarer/valera/workflows/Unit%20Tests%20&%20Lint/badge.svg)


## Install

```bash
gem install bundler
bundle install
```
## Run

```bash
ruby main.rb
```


## Run test

```bash
rspec
```

## Run rubocop

```bash
rubocop
rubocop -a # Auto-correct offenses
```
